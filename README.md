# scala-shop
A shop system designed in Scala

## Requirements
* [Java](https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html)
* [Scala](https://www.scala-lang.org/download/)
* [sbt](https://www.scala-sbt.org/download.html)
